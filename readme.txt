== Description ==

<strong>Plugin for Gear5 website performance measurement and alerting system</strong>
<ul>
<li>Plugin enables you to easily setup Gear5 performance measurement tool.</li>
<li>Gear5 is an free, easy-to-use, website performance monitoring and alerting system that will give you comprehensive performance information to help you speed up your websites.</li>
</ul>
<strong>Using Gear5 you will be able to:</strong>
<ul>
<li>Monitor performance of your website(s) in real time</li>
<li>Receive alerts when your website is facing slow performance or complete outage</li>
<li>Integrate performance data with Google Analytics metrics</li>
<li>Get monthly reports for key website performance indicators</li>
<li>Analyse load times broken down by tires, countries, browsers and user defined pages</li>
<li>To get more info about Gear5 and to get your free account, please visit and register at <a href="http://www.gear5.me">http://www.gear5.me</a></li>
</ul>

== Installation ==

1. Upload `gear5.zip` onto your local computer.
2. Go to your WordPress Dashboard and select <strong>Plugins >> Add New</strong>.
3. Click on the <strong>Upload</strong> option at the top and select the `gear5.zip` file you just downloaded.
4. Click on <strong>Install</strong>.
5. Activate the plugin through the 'Plugins' menu in WordPress
6. Press Settings or go to Setting->Gear5
7. Configure plugin by entering your Gear5 API key.

To get API key:

1. Register and add your website at http://www.gear5.me 
2. Go to Gear5 Dashboard and click "Edit" for your web site
3. On top of General setting your api key will be displayed.
4. Copy and paste your API key into Gear5 WordPress plugin settings and press "Save Options".
5. You are set up!
